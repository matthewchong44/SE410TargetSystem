About

The target system is an open-source toolkit for educational resources. This toolkit contains a great amount of collections online, providing access to scanned versions of educational material, and links to external sources. These collections consist of modules, each composed of various materials to organize such data. These materials include, data sets, practice problems, lab exercises, etc. These toolkits allow users to create, access, and export the collections of the education resources. Whether the user is a student, professor, teaching-assistant, or scholar, the target system is the go-to place to explore and read thousands of free, educational materials.

The project is going to focus on forward engineering the existing systems for SWENET and EdRepo into a new target system. The target system will provide a user-friendly digital library for people who are using the target system. Rebuilding the open-source digital library to share with the public also allows the community to get involved in improving the existing library.
